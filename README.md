**Telecom Customer Churn Analysis**

**Project Overview:**

Customer churn is a major challange for telecom companies. This project focuses on analysing customer churn behaviour to identify the key factors that influence customer attrition. The goal is to derive actionable insights that can help telecom businesses improve customer retention strategies.

The project uses **Excel, SQL, Python, and Power BI** to perform end-to-end data analysis, including data exploration, visualization, and dashboard development.

**Objectives:**

1. Analyse customer churn patterns in a telecom dataset.
2. Identifying factors influencing customer churns.
3. Perform Exploratory data analysis (EDA) using Python.
4. Create interactive dashboards using Power BI.
5. Generate insights that can help reduce customer churn.

**Tools:**

1. **Excel:** Data exploration, pivot tables, and charts.
2. **SQL:** Data querying and churn-related analysis.
3. **Python:** Exploratory Data Analysis using Pandas, Matplotlib, and Seaborn.
4. **Power BI:** Interactive dashboard and business insights.
5. **Github:** Project documentation and version control.

**Project Workflow:**

**Data Collection:**

Telecom customer churn dataset containg customer demographics, services, charges, and churnnstatus.

**Data Exploration (Excel):**

1. Created pivot tables and charts.
2. Analysed churn distribution across different customer segments.

**SQL Analysis:**
1. Wrote queries to analyse churn patterns.
2. Identified trends in customer behaviour and service usage.

**Python Exploratory Data Analysis:**

1. Data cleaning and transformation.
2. Visualization of churn trends.
3. Correlation and feature analysis.

**Power BI Dashboard:**

1. Developed an interactive churn analysis dashboard.
2. Visualsed churn distribution, churn reasons, and customer demographics.

**Key Insights:**

1. Customers with **month-to-month contracts** have a higher churn rate.
2. **Higher monthly charges** are associated with increased churn probability.
3. **Competitor offers and pricing** are major reasons for customer churn.

**Dashboard Features:**

Ther Power BI dashboard provides insights into:
1. Total customers and churn rate.
2. Churn vs retained customers.
3. Churn reasons and categories.
4. Customer demographics (gender, contact type, payment method).
5. Geographic distribution of customers.

**Dataset:**

  The dataset includes information about telecom customers such as:
  1. Customer demographics
  2. Internet services
  3. Contract types
  4. Monthly charges
  5. Payment methods
  6. Customer churn status

**Business Impact:**

  Understanding customer churn helps telecom companies:
  1. Improve customer retention strategies
  2. Identify high-risk customer segments
  3. Optimize pricing and service offerings
  4. Enhance overall customer satisfaction
